# Dapoer Papoea

Dapoer Papoea adalah proyek open-source untuk mendokumentasikan dan mempopulerkan resep kuliner khas Papua. Tersedia secara bebas agar siapa saja dapat mengeksplorasi, mencoba, dan turut melestarikan kekayaan masakan Papua.

## Fitur

- Koleksi resep makanan pokok, lauk, camilan, dan minuman khas Papua
- Navigasi antar resep dalam satu kategori
- Antarmuka yang responsif untuk semua ukuran layar

## Kontribusi

Kami menyambut kontribusi dari siapa saja, baik berupa penambahan resep, perbaikan konten, maupun peningkatan teknis. Baca [CONTRIBUTING.md](./CONTRIBUTING.md) untuk panduan lengkapnya.

## Kredit

Proyek ini dikembangkan oleh [Papua Open Source](https://papuaopensource.github.io), komunitas teknologi pemuda-pemudi Papua yang berkomitmen memajukan Papua melalui inovasi digital.

Seluruh konten resep bersumber dari buku **"Resep Masak Dapur Irian Jaya"**, disusun oleh Marijke Werimon-Bakker, Hendrina Rumsaur, Dianne Gee, dan Reggie Pangkatana, diterbitkan oleh Pusat Latihan Pelayan Sosial GKI Padang Bulan, Abepura.
